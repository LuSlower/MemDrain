```
  -ws <IM> <-r>           | Drain WorkingSet
  -sws                    | Drain SystemWorkingSet
  -mpl                    | Drain ModifiedPageList

  -sl <0>                 | Drain StanbyList (and low priority)

  -all                    | Drain All
```
